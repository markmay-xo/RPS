# Barbershop Waitlist

## Problem

Design a waitlist system for a single-chair barbershop.

There is one barber, one chair, and customers walk in the door. The barber needs to:

- Add a customer to the waitlist when they walk in
- See who's next
- Remove the current customer when their cut is done
- Tell a walk-in roughly how long the wait is (average cut is 20 minutes)
